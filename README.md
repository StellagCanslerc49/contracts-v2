# Hop

The smart contracts that power the Hop Exchange.

## Compile Contracts
```
npm run compile
```

## Test
```
npm run test
```

## Getting Started
Create a `.env` file based on `.env-sample`

## Deploy Local
Start local node
```
npm run chain
```

Deploy to local node
```
npm run deploy:local
```

## Deploy
Deploy to Goerli
```
npm run goerli
```

Deploy to Mainnet
```
npm run mainnet
```
